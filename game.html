Facts Game About App

This is an interactive game that uses the intellect of a user in order to achieve the maximum amount of points. This game is created using JavaScript. The main objective is to stay away from the other circles, that represent carriers of the Covid-19 virus. The overall goal of this game is to show the purpose of why social distancing matters as you will become a carrier of the Covid-19 virus.
